# English to French Translator (LangChain + Gemini)

A simple Streamlit-based language translation app built to learn and explore
LangChain, Google Gemini API, and LangSmith monitoring.

## Features
- English to French translation using Gemini 1.5 Flash
- LangChain prompt pipelines
- Streamlit UI
- Environment-based configuration
- LangSmith tracing and monitoring

## Tech Stack
- Python
- LangChain
- Google Gemini API
- Streamlit
- LangSmith
